Comment ça marche

Simple comme bonjour.

01

On discute

Vous nous expliquez votre activité, vos attentes, vos goûts. Un échange rapide suffit — pas de réunion de 2h.

02

On conçoit

On propose une maquette. Vous validez ou suggérez des retours. On itère jusqu'à ce que ce soit parfait pour vous.

03

On code

On développe le site en HTML/CSS/JS. Code propre, rapide, sans dépendances inutiles. Zéro bloat.

04

On publie

Votre site est en ligne en quelques minutes. Vous recevez le lien, les accès, et on reste disponibles si besoin.

🔗 L'hébergement, c'est gratuit pour vous.

On utilise GitHub Pages pour héberger votre site sans frais. Votre site sera accessible sur une URL du type votreentreprise.github.io. Si vous souhaitez un domaine personnalisé comme votreentreprise.fr, c'est vous qui le louez (~10€/an), et on fait la liaison.
